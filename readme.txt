FIBER-E1-PWR
Плата питания преобразователя ОПТИКА - E1.

Плата предназначена для формирования напряжений 5 В из 220 В переменного и постоянного напряжения и 5 В из 48 В постоянного напряжения.
Предохранители по 220В 520.609 на ток 0.160А, по 48В 520.611 на ток 0.250А.
Держатели под предохранители 5х20 на панель 601.100 ESKA.

01v1

Код ПП: 3.140.202
Код ПП в сборе: 2.140.237

Проблемы:
1. Для VD5,VD6 маленькое посадочное место. 
2. На сборочнике надо убрать треугольники полярности у диодов VD5,VD6.

Изменения:
1. Добавлена цепь заряда конденсаторов.
2. В стэке медь увеличина до 35 мкм. 
3. Крепежные отверстия для соединения с землей заменены на прошитые переходными отверстиями.
4. Предохранители использовать на 160 мА и 250 мА.
